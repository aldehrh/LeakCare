## 서버 실행 방법
---
### 1. git 복제 (cmd에서 진행)
```git clone https://github.com/hyun337/LeakCare.git```
<br />

```cd Leakcare```

<br />

### 2. 경로 이동
```cd backend```

<br />

### 2. 패키지 설치
```pip install -r requirements.txt```

<br />

### 3. 서버 실행
```uvicorn app.main:app --reload```

<br />

### 4. 주소 접속
```http://127.0.0.1:8000/docs```
